React Native 學習筆記

發表於 2016-12-14

  1. 在建立新專案時可指定 react-native module 版本
    範例：react-native init FirstProject –version react-native@0.38.1

  2. [RN-Android] 使用 react-native run-android --variant={floversName} 命令，其中 --variant 可以指定要編譯的flovers名稱
    範例：flovers有 googlePlay, amazonStore

  3. [RN-Android] 使用 react-native run-android --configuration Release 命令，可編譯apk為release並安裝到裝置上

  4. [RN-Android] 開啟Android專案必須手動新增main底下的assets資料夾，否則run-android –configuration Release會失敗…

  5. 手動下載index.android.bundle檔 stackoverflow

  6. [RN-Android] react-native-cli：1.0.0、react-native：0.39.2
    無法使用 react-native run-android --configuration Release 問題

    • android 要自己建立assets資料夾 android/app/src/main/assets
    • 自行建立keystroe，開啟android studio build->generate signed apk…
    • 確認release.jks存在android/release.jks

    • 開啟 android/app/build.gradle

      即可使用react-native run-android –configuration Release 來編譯不需要render server的apk
